# ML Projects Using Kaggle Datasets

This repository contains **two Jupyter Notebooks** for machine learning projects using Kaggle datasets:
1. **Classification of NASA Near-Earth Objects**
2. **Prediction of Medical Insurance Costs**

Both projects include **data preprocessing, model training, evaluation, and visualization**.

---

## Projects Overview

### 1. **NASA Near-Earth Objects Classification**
**Dataset**: [NASA NEO (1910-2024)](https://www.kaggle.com/datasets/ivansher/nasa-nearest-earth-objects-1910-2024)
**Notebook**: [`notebooks/classification.ipynb`](notebooks/classification.ipynb)

#### Key Steps:
- **Data Cleaning**: Handling missing values, outliers, and feature selection.
- **Exploratory Analysis**: Visualizing distributions.
- **Model Training**: Logistic Regression, Random Forest, XGBoost.
- **Evaluation**: Accuracy, Precision, Recall, F1-Score, Confusion Matrix.

---

### 2. **Medical Insurance Cost Prediction**
**Dataset**: [Medical Insurance Cost](https://www.kaggle.com/datasets/mohankrishnathalla/medical-insurance-cost-prediction)
**Notebook**: [`notebooks/medical_insurance.ipynb`](notebooks/medical_insurance.ipynb)

#### Key Steps:
- **Data Preprocessing**: Encoding categorical variables, scaling numerical features.
- **Feature Engineering**: Creating interaction terms for better model performance.
- **Model Training**: Linear Regression, Random Forest, Gradient Boosting.
- **Evaluation**: MAE, RMSE, R² Score.

---

## Techniques Used
- **Libraries**: Pandas, NumPy, Scikit-learn, XGBoost, Matplotlib, Seaborn.
- **Preprocessing**: Normalization, One-Hot Encoding, Train-Test Split.
- **Model Evaluation**: Cross-validation, performance metrics, visualization.
- **Visualization**: Histograms, Box Plots, Feature Importance.
